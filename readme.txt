    В данном проэкте 2 основных файла: test_all.py и main_page.py.
test_all.py - отвечает за порядок запуска тестов и вспомогательных методов.
main_page.py - содержит код тестов и вспомогательных методов.

    Данные для тестирования в настоящий момент хранятся в словаре находящемся в начале файла main_page.py и используются
как для заполнения начальной формы так и для проверки данных, данные прокоментированны около словаря. Так же данные
сохраняются в файл 'static/data/stedent.txt' в формате JSON.
    Для заполнения одного из полей требуется файл с картинкой который храниться в 'static/image/dtjxK5Y9Bt8.jpg'.
    При заполнение и проверке полей с календарём использован файл с подстановочными данными содержащий
     отобрание коротких названий месяцев их длинные названия 'static/data/data.py'.
    Логгер Аллюре пишет данные в папку 'allure-results' для этого используется команда pytest --alluredir allure-results
    Для запуска веб сервера в Windows с отбражением статистики используется команда:
    'allure serve <путь к папке с отчётами>'
